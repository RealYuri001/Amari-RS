# Amari.rs

An easy to use API wrapper for AmariBot in Rust language.

## Minimum Supported Rust Version (MSRV)

Rust 1.65.0 or later is required for compability.

## Installation

Simply by adding the dependency in Cargo.toml:
```
[dependencies]
amari-rs = "0.1.0"
```

Or by using:
```
cargo add amari-rs
```
